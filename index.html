1.0 Introduction

The Wizzer Works Toolroot is a collection of Makefiles and tools which facilitate cross-platform development. The Makefiles are designed to support both the C and C++ programming languages. However, there is no reason why this methodolgy could not be extended to other programming languages.

The current version of the Wizzer Works Toolroot supports the Linux and Windows development platforms. However, this version has not yet been tested on all flavors of these operating systems. Version 0.5 of the Wizzer Works Toolroot supports:

  • Ubuntu 18.04 LTS 64-bit
  • Windows 10

To use this tool suite effectively, some guidelines should be followed in the way a project is structured. These guidelines are discussed below in Section 4.0, Makefile Style Guide, and Section 5.0, Software Architecture Style Guide.

3.0 Wizzer Works Tools

The Wizzer Works Toolroot is currently a collection of Makefiles and the InstallBuilder tools. The Makefiles are GNU make compliant makefiles. The InstallBuilder tools are three Perl scripts used for generating installation packages.

The Makefiles are:

  • commondefs - common Makefile definitions
  • commonrules - common Makefile rules
  • configurations - project configurations (i.e. debug and release)
  • installdefs - installation Makefile definitions
  • installrules - installation Makefile rules
  • installversiondefs - versioning installation Makefile definitions
  • installversionrules - versioning installation Makefile rules
  • ismcommondefs - ISM common Makefile defintions
  • ismcommonrules - ISM common Makefile rules
  • mandefs - man documentation Makefile definitions
  • manrules - man documentation Makefile rules

The Install Builder tools are:

  • ibCollect.pl - a Perl script for generating a list of files to be included in a distribution package
  • ibVerifyCollection.pl - a Perl script for verifying the collection of files to be included in a distribution package against a master list
  • ibBuild.pl - a Perl script for generating a distribution package
